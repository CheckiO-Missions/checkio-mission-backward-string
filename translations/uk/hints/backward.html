Давай проведемо маленький тест. Що буде, якщо ти відкриєш свою консоль python та впишеш "a = [1, 2, 3]" (без лапок ""), а потім використаєш "негативний крок" (negative step) до цього списку?

Що таке негативний крок?

Ти можеш використати "негативний крок" до списку, щоб отримати реверсивну версію списку:

a = [1, 2, 3]
a[::-1]

У відповідь ти отримаєш список у вигляді [3, 2, 1].

Останнє запитання. Як має виглядати мій фінальний код? 
def backward_string(val: str) -> str:
    # your code here
    return val[::-1]
