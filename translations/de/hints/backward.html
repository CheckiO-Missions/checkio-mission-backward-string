Lassen Sie uns einen einfachen Test machen. Was passiert, wenn Sie Ihre Python-Konsole öffnen und "a = [1, 2, 3]" sagen und dann einen negativen Schritt auf diese Liste anwenden?

Was ist ein "negativer Schritt"?

Sie können einen negativen Schritt verwenden, um eine umgekehrte Liste zu erhalten:

a = [1, 2, 3]
print(a[::-1])

Ausgabe:

[3, 2, 1]

Eine letzte Frage. Wie wird meine endgültige Lösung aussehen? 

def backward_string(val: str) -> str:

    return val[::-1]
