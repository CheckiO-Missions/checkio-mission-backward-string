Faisons un test simple. Que se passera-t-il si vous ouvrez votre console Python et dites "a = [1, 2, 3]" et que vous utilisez ensuite un pas négatif pour cette liste ?

Qu'est-ce qu'un "pas négatif" ?

Vous pouvez utiliser un pas négatif pour obtenir une liste inversée :

a = [1, 2, 3]
print(a[::-1])

Résultat :

[3, 2, 1]

Une dernière question. A quoi ressemblera ma solution finale ? 

def backward_string(val : str) -> str :

    return val[::-1]
