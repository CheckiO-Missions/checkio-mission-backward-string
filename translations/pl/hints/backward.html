Proszę zrobić prosty test. Co się stanie, jeśli otworzy Pan konsolę Pythona i powie "a = [1, 2, 3]", a następnie użyje ujemnego kroku do tej listy?

Co to jest "krok ujemny"?

Można użyć kroku ujemnego, aby uzyskać odwróconą listę:

a = [1, 2, 3]
print(a[::-1])

Wynik:

[3, 2, 1]

Ostatnie pytanie. Jak będzie wyglądało moje ostateczne rozwiązanie? 

def backward_string(val: str) -> str:

    return val[::-1]
