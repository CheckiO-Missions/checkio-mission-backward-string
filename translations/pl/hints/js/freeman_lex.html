Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

pomysł polega na zadeklarowaniu nowej zmiennej typu string, iterowaniu po value i dodawaniu każdego znaku na początku nowego ciągu.

Ok, od czego powinienem zacząć?

Zadeklarujmy nową zmienną res z wartością początkową pustego ciągu. 

let res = "";

To proste! Ale jak przenieść początkowy ciąg znaków?

W tym celu proszę użyć pętli for/of. 

let res = "";
for (let char of value)
    ...;

Ponieważ wewnątrz pętli będzie tylko jedna linia kodu, można nie używać nawiasów klamrowych.

Nie rozumiem, jak prawidłowo dodać każdy znak do nowego ciągu.

Musi Pan otrzymać ciąg znaków w kolejności wstecznej. Jeśli przy normalnej kolejności wystarczy dodać nowy znak na końcu istniejącej linii znaków, to przy kolejności wstecznej należy dodać każdy nowy znak przed istniejącą linią. 

let res = "";
for (let char of value)
    res = char + res;

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć swój kod w wyrażenie funkcji i zwrócić nowy ciąg res.

Oto całe rozwiązanie:

function backwardString(value: string): string {

    let res = "";
    for (let char of value)
        res = char + res;
    
    return res;
}

To rozwiązanie wykorzystuje rekurencję - za każdym razem proszę umieścić pierwszy znak po wywołaniu tej samej funkcji, ale z mniejszym argumentem string. Aby uniknąć nieskończonej rekurencji, proszę po prostu zwrócić ciąg od jednego znaku. 

function backwardString(value: string): string {

    if (value.length < 2)
        
        return value;
    
    return backwardString(value.slice(1)) + value.charAt(0);
}

Rozwiązanie jednoliniowe, w którym dzielimy początkowy ciąg przez pusty znak (wynik - tablica wszystkich znaków), odwracamy tablicę i łączymy wszystkie znaki z pustym znakiem między każdym z nich - po prostu łączymy wszystkie znaki w nowym ciągu. 

function backwardString(value: string): string {
    
    return value.split('').reverse().join('');
}
