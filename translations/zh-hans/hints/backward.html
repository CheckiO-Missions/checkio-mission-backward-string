让我们做一些简单的测试。如果你打开你的Python控制台，说 "a = [1, 2, 3]"，然后对这个列表使用一个负数步骤，会是什么情况？

什么是 "负数"？

你可以使用负数步长来获得一个反转的列表：

a = [1, 2, 3]
print(a[::-1])

输出：

[3, 2, 1]

最后一个问题。我的最终解决方案会是什么样子的？ 

def backward_string(val: str) -> str:

    return val[::-1]
