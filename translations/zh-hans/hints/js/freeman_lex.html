让我们声明新的变量res ，初始值为空字符串。 

let res = "";

这很简单!但如何转移初始字符串呢？

为此目的，使用for/of 循环。 

let res = "";
for (let char of value)
    ...;

由于你在循环内只有一行代码，你可以不使用弯括号。

我不明白如何将每个字符正确地添加到新字符串中。

你需要得到一个具有倒序字符的字符串。如果是正常的顺序，你只需将新的字符添加到现有的字符行的末尾，那么在倒序的情况下，你需要将每个新的字符添加到现有的行之前。 

let res = "";
for (let char of value)
    res = char + res;

那么，我的最终解决方案应该是什么样子的呢？

你只需要将你的代码包装成函数表达式并返回新的字符串res 。

以下是整个解决方案：

function backwardString(value: string): string {

    let res = "";
    for (let char of value)
        res = char + res;
    
    return res;
}

这个解决方案使用了递归--每次你都把第一个字符放在同一个函数的调用之后，但用较小的字符串参数。为了避免无限递归，只需从一个字符返回字符串。 

function backwardString(value: string): string {

    if (value.length < 2)
        
        return value;
    
    return backwardString(value.slice(1)) + value.charAt(0);
}

单行解决方案，即用空字符分割初始字符串（结果--所有字符的数组），反转数组并连接所有字符，每两个字符之间为空字符--简单地连接新字符串中的所有字符。 

function backwardString(value: string): string {
    
    return value.split('').reverse().join('');
}
