I don't understand how to add each character to the new string properly.

You need to get a string with backward order of characters. If with normal order you just add new character to the end of the existed line of characters, so with backward order you need to add each new character before the existed line. 

let res = "";
for (let char of value)
    res = char + res;
So, how my final solution should look like?

You just need to wrap your code into function expression and return new string res.

Here is the whole solution:

function backwardString(value: string): string {

    let res = "";
    for (let char of value)
        res = char + res;
    
    return res;
}

This solution uses recursion - each time you put first character after the call of the same function, but with smaller string argument. To avoid infinite recursion, just return the string from one character. 

function backwardString(value: string): string {

    if (value.length < 2)
        
        return value;
    
    return backwardString(value.slice(1)) + value.charAt(0);

One-line solution, where you split the initial string by the empty character (result - Array of all characters), reverse the Array and join all characters with empty character between each two - simply join all characters in the new string. 

function backwardString(value: string): string {
    
    return value.split('').reverse().join('');
}
